Alacer Offerings

Data Cloaker

Anonymize data easily with the Alacer Data Cloaker. The Cloaker allows users mask or remove confidential, sensitive, or private information from columns in a .csv file. The Data Cloaker exports separate .csv files for the "cloaked" data and for the appropriate keys illustrating the masked values.

Data Preparer

Data preparation includes cleaning, managing, and sharing high-quality data. Alacer offers customized data quality management services that will allow your company to share high-quality data within-house and with others.

Data Analyzer

Alacer produces cutting-edge data analysis and results presentation. We routinely assist clients with big data, machine learning, predictive analytics, and visualization solutions. Our drill-down dashboard is one of several customizable analysis outputs that we can create to fit your needs.

Data Uncloaker

Once analysis stages are complete, the Data Uncloaker allows one to recover the unmasked data values, even if the analysis was completed using the cloaked data. Our interface creates a seamless way to transform cloaked values back into their original data.
